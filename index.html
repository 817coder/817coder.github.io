Hexo常用命令

发表于 2013-05-13 更新于 2021-05-13 分类于 hexo

new

新建一篇文章

此时 Hexo 会创建 source/_posts/title.md，并采用默认的 layout。

generate

生成静态文件。

选项 描述
-d, –deploy 文件生成后立即部署网站

该命令可以简写为

publish

发表草稿。

server

选项 描述
-p, --port 重设端口
-s, --static 只使用静态文件
-l, --log 启动日记记录，使用覆盖记录格式

deploy

部署网站。

参数 描述
-g, --generate 部署之前预先生成静态文件

该命令可以简写为：

clean

清除缓存文件 (db.json) 和已生成的静态文件 (public)。

在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。

显示草稿

显示 source/_drafts 文件夹中的草稿文章。
